/*
  Computa los valores y elige el más bajo
  Si el 98% de la pantalla es < que 680px coge este valor
  En caso contrario elige 680px

  Lo que es equivalente a:

  width: 98%;
  max-width: 680px;
*/
.min {
  /*
  Se pueden pasar múltiples valores siempre que vayan separados por comas
  width: min(98%, 680px + 10vw, 1000px, 20em);
  */
  width: min(98%, 680px);
}

/*
  Computa los valores y elige el más alto
  Si el 98% de la pantalla es > que 680px coge este valor
  En caso contrario elige 680px

  Lo que es equivalente a:

  width: 98%;
  min-width: 680px;
*/
.max {
  width: max(680px, 98%);
}

/*
  clamp(MIN, VAL, MAX) puede resolverse como
  max(MIN, min(VAL, MAX))

  Por lo tanto estaríamos indicando un valor mínimo con MIN que no podría reducir
  Un valor máximo con MAX que no se podría sobrepasar
  Un valor ideal entre ambos

  Esta fórmula asegura que se aplique el valor más grande entre un mínimo y el más pequeño de entre uno ideal y uno máximo
  Si VAL es superior a MAX, se descarta porque seleccionamos el mínimo, y se va a resolver con MIN escogiendo el valor más alto
*/
.clamp {
  width: clamp(320px, 90%, 48rem);
}

/*
  Se pueden combinar estas funciones entre ellas
  En este ejemplo buscamos un valor mínmio seguro que sería 1rem
  Si la obtención de un valor con la funcioónes min() y calc() resultase 
  en un valor < que 1rem no sería elegido y por tanto 1rem actuaría como fallback

  Si aumentamos el tamaño de la pantalla el cálculo que hacemos en max()
  supera el valor 1rem y entonces es escogido

  El primer valor es obtenido mediante min(), y será el menor de entre los tres valores 
  que le pasamos. 
*/
p {
  font-size: calc(1em + 0.25vw);
  font-size: max(min(10%, ((1.25vw + 10px) * 0.5), 5vw), 1rem);
  font-size: max(min(10%, calc((1.25vw + 10px) * 0.5), 5vw), 1rem);
}

p:not(:last-child) {
